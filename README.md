# 🚀 KidoAssembly
**KidoAssembly** is an interactive, gamified learning platform designed to make education fun and engaging for students in Grades 1 to 5. Through a combination of mini-games, virtual pets, and a rewarding progression system, KidoAssembly encourages continuous learning and exploration.
🌐 **Live Demo:**
https://kiddoassembly.netlify.app/
## ✨ Features
 * 🎮 **Educational Mini-Games:** * **Math Shooter:** Action-packed math problem solving.
   * **Word Scramble:** Boost vocabulary and spelling skills.
   * **Memory Match:** Enhance cognitive memory through fun card matching.
   * **Interactive Quizzes:** Test knowledge across various subjects.
 * 🏆 **Gamified Progression:** Students earn **Stars, XP, and Energy** by completing learning quests and daily missions.
 * 🐾 **Virtual Pet Center:** Keep students engaged by allowing them to adopt, feed, and play with their own virtual pet using the rewards they earn.
 * 🛒 **Playground Shop:** Spend hard-earned stars to unlock exciting new items in the digital playground.
 * 👩‍🏫 **Teacher Dashboard:** A dedicated space for educators to monitor progress and manage content.
 * 👤 **Customizable Profiles:** Students can set up their avatars, track their grade level, and view their completed quests.
## 🛠️ Tech Stack
 * **Frontend:** React.js
 * **Styling:** Tailwind CSS
 * **Backend & Database:** Firebase (Firestore & Authentication)
 * **Hosting:** Netlify
